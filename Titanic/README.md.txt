# Анализ выживаемости пассажиров Титаника: EDA и модели машинного обучения

## Описание проекта

В рамках проекта была проведена разведывательная аналитика и построены модели машинного обучения с целью предсказания выживаемости пассажиров на основе исторических данных о крушении Титаника. Работа включает визуализацию данных, создание новых признаков и сравнение различных алгоритмов классификации.

## Этапы исследования

### 1. Разведывательный анализ данных (EDA)

- Изучены распределения числовых признаков (`Age`, `Fare`)
- Проведен анализ категориальных переменных (`Sex`, `Pclass`, `Embarked`) и их взаимосвязь с целевой переменной `Survived`
- Построена корреляционная матрица для количественной оценки связей между признаками

### 2. Feature Engineering

- Извлечена информация о социальном статусе пассажиров (`Title`) из поля `Name`
- Обработаны пропущенные значения в числовых и категориальных колонках
- Признаки были приведены к числовому виду для обучения моделей
- С помощью случайного леса оценена важность новых и исходных признаков

### 3. Обучение моделей

Были реализованы и протестированы модели из разных семейств:
- Логистическая регрессия (линейный классификатор)
- Дерево решений
- Градиентный бустинг (XGBoost)
- Простая нейронная сеть (на базе Keras)

Для оценки качества использовалась стратифицированная кросс-валидация на 5 фолдах.

## Используемые библиотеки

- pandas, numpy — для работы с данными
- seaborn, matplotlib — для визуализации
- scikit-learn — для моделей и кросс-валидации
- xgboost — бустинг
- keras / tensorflow — нейронная сеть

## Вывод

Наилучшие результаты показала модель XGBoost, что делает её предпочтительным выбором для задачи. Также достойную точность показала нейронная сеть, однако её обучение потребовало больше вычислительных ресурсов.
